## Technical Debt Inventory

1. **Unused `d3-force` dependency** — ~30KB, never imported
2. **No Zod validation on parsed output** — schema exists but `parseKG` doesn't use it
3. **`alert()` in 2 locations** — `use-file-drop.ts:22`, `Toolbar.tsx:73`
4. **`setTimeout` for focus** — `SearchPanel.tsx:19` — race condition
5. **No input validation on "Add Field"** — can create field named `id`
6. **`computeWidth` recursive, no depth limit** — stack overflow risk
7. **No debounce on search** — lag at 10K nodes
8. **`ConnectToRow` renders all nodes in `<select>`** — 1000+ option dropdown
9. **`proOptions: { hideAttribution: true }`** — possible React Flow license violation
10. **Theme set on `document.documentElement`** — conflicts when multiple instances embedded
11. **`handleNodeJsonSave` batching** — creates N history entries instead of 1
12. **Missing `type="button"` on buttons** — may submit forms when embedded
13. **`"excalidraw"` in npm keywords** — misleading
